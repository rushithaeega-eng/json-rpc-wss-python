# Secure JSON-RPC 2.0 over WebSocket (WSS)

## 🚀 Project Overview
This project implements secure communication using JSON-RPC 2.0 over WebSocket Secure (WSS) with Python.

## 🛠 Technologies Used
- Python
- WebSockets
- JSON-RPC 2.0
- SSL/TLS

## ▶️ How to Run

### Install dependencies
pip install websockets

### Run Server
python server.py

### Run Client
python client.py
